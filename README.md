# Chileleko Lombe Moonga — Portfolio

Multi-page professional portfolio site.

## Pages
- `index.html` — Home
- `about.html` — About
- `projects.html` — Projects
- `resume.html` — Resume & Certifications
- `impact.html` — Impact & Skills
- `contact.html` — Contact
- `shared.css` — Shared styles (used by all pages)

## To add your resume PDF
1. Create a `docs/` folder in this repo
2. Upload your resume PDF as `docs/resume.pdf`
3. The Download Resume button on the Resume page will link to it automatically

## Deploy to Vercel
1. Push all files to a GitHub repo
2. Go to vercel.com → Add New Project → Import repo
3. Leave all settings as default → Deploy

Any `git push` will auto-redeploy.
